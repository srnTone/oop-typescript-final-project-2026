## โมเดล Service หรือข้อมูลบริการ

id              ประเภท string   เป็นรหัสสากลอ้างอิงบริการที่ไม่ซ้ำกัน

name            ประเภท string   เป็นชื่อของบริการที่เปิดให้จอง

description     ประเภท string   เป็นรายละเอียดและเงื่อนไขการรับบริการ

price           ประเภท number   เป็นราคาค่าบริการซึ่งต้องมีค่าไม่ติดลบ

durationMinutes ประเภท number   เป็นระยะเวลาที่ใช้ในการให้บริการมีหน่วยเป็นนาที

category        ประเภท string   เป็นหมวดหมู่ของบริการเพื่อใช้กรองข้อมูล

status          ประเภท ServiceStatus Enum เป็นสถานะของบริการว่าพร้อมให้บริการหรือปิดปรับปรุง

isActive        ประเภท boolean  เป็นตัวกำหนดการมองเห็นของลูกค้าบนระบบ

createdAt       ประเภท Date     เป็นวันและเวลาที่บันทึกข้อมูลบริการนี้เข้าระบบ

updatedAt       ประเภท Date     เป็นวันและเวลาที่แก้ไขข้อมูลบริการล่าสุด

---

## โมเดล Appointment หรือข้อมูลการนัดหมาย

id              ประเภท string   เป็นรหัสสากลอ้างอิงการจองที่ไม่ซ้ำกัน

serviceId       ประเภท string   เป็นรหัสบริการที่ลูกค้าต้องการจองเพื่อเชื่อมโยงข้อมูล

customerName    ประเภท string   เป็นชื่อนามสกุลของลูกค้าผู้ทำรายการ

customerEmail   ประเภท string   เป็นอีเมลของลูกค้าสำหรับส่งข้อมูลยืนยัน

customerPhone   ประเภท string   เป็นเบอร์โทรศัพท์ติดต่อของลูกค้า

appointmentDate ประเภท Date     เป็นวันที่ลูกค้าต้องการเข้ามารับบริการ

startTime       ประเภท string   เป็นเวลาที่เริ่มต้นรับบริการ

status          ประเภท AppointmentStatus Enum เป็นสถานะของการนัดหมายเช่นรอยืนยันหรือเสร็จสิ้น

notes           ประเภท string   เป็นหมายเหตุหรือคำขอพิเศษเพิ่มเติมจากลูกค้า

createdAt       ประเภท Date     เป็นวันและเวลาที่ลูกค้ากดทำรายการจอง
