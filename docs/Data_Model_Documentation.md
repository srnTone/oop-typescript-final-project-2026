# Data Model Documentation

## โมเดล Service หรือข้อมูลบริการ

### 1) id
- ประเภท: `string`
- คำอธิบาย: เป็นรหัสสากลอ้างอิงบริการที่ไม่ซ้ำกัน

### 2) name
- ประเภท: `string`
- คำอธิบาย: เป็นชื่อของบริการที่เปิดให้จอง

### 3) description
- ประเภท: `string`
- คำอธิบาย: เป็นรายละเอียดและเงื่อนไขการรับบริการ

### 4) price
- ประเภท: `number`
- คำอธิบาย: เป็นราคาค่าบริการ ซึ่งต้องมีค่าไม่ติดลบ

### 5) duration
- ประเภท: `number`
- คำอธิบาย: เป็นระยะเวลาที่ใช้ในการให้บริการ มีหน่วยเป็นนาที

### 6) category
- ประเภท: `string`
- คำอธิบาย: เป็นหมวดหมู่ของบริการเพื่อใช้กรองข้อมูล

### 7) provideName
- ประเภท: `string`
- คำอธิบาย: เป็นชื่อของผู้ให้บริการที่รับผิดชอบบริการนั้นๆ

### 8) status
- ประเภท: `ServiceStatus` (Enum)
- คำอธิบาย: เป็นสถานะของบริการว่าพร้อมให้บริการหรือปิดปรับปรุง

### 9) isActive
- ประเภท: `boolean`
- คำอธิบาย: เป็นตัวกำหนดการมองเห็นของลูกค้าบนระบบ

### 10) createdAt
- ประเภท: `Date`
- คำอธิบาย: เป็นวันและเวลาที่บันทึกข้อมูลบริการนี้เข้าระบบ

### 11) updatedAt
- ประเภท: `Date`
- คำอธิบาย: เป็นวันและเวลาที่แก้ไขข้อมูลบริการล่าสุด

## โมเดล Appointment หรือข้อมูลการนัดหมาย

### 1) id
- ประเภท: `string`
- คำอธิบาย: เป็นรหัสสากลอ้างอิงการจองที่ไม่ซ้ำกัน

### 2) serviceId
- ประเภท: `string`
- คำอธิบาย: เป็นรหัสบริการที่ลูกค้าต้องการจองเพื่อเชื่อมโยงข้อมูล

### 3) customerName
- ประเภท: `string`
- คำอธิบาย: เป็นชื่อนามสกุลของลูกค้าผู้ทำรายการ

### 4) customerEmail
- ประเภท: `string`
- คำอธิบาย: เป็นอีเมลของลูกค้าสำหรับส่งข้อมูลยืนยัน

### 5) customerPhone
- ประเภท: `string`
- คำอธิบาย: เป็นเบอร์โทรศัพท์ติดต่อของลูกค้า

### 6) appointmentDate
- ประเภท: `Date`
- คำอธิบาย: เป็นวันที่ลูกค้าต้องการเข้ามารับบริการ

### 7) startTime
- ประเภท: `string`
- คำอธิบาย: เป็นเวลาที่เริ่มต้นรับบริการ

### 8) status
- ประเภท: `AppointmentStatus` (Enum)
- คำอธิบาย: เป็นสถานะของการนัดหมาย เช่น รอยืนยัน, ยืนยันแล้ว, เสร็จสิ้น, หรือยกเลิก

### 9) notes
- ประเภท: `string`
- คำอธิบาย: เป็นหมายเหตุหรือคำขอพิเศษเพิ่มเติมจากลูกค้า

### 10) createdAt
- ประเภท: `Date`
- คำอธิบาย: เป็นวันและเวลาที่ลูกค้ากดทำรายการจอง

### 11) updatedAT
- ประเภท: `Date`
- คำอธิบาย: เป็นวันและเวลาที่แก้ไขข้อมูลนัดหมายล่าสุด
